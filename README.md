# Tomato_Timer
A work &amp; study timer

# Features
* Icon Badge shows how much time you left to finish the current session
* Customize the length of Focus Session and short break

# Usage
* ```git clone https://github.com/xiaoluo2017/Tomato_Timer```
* open index.html
